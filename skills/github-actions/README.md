# GitHub Actions

Human-facing overview for the `github-actions` skill.

## What It Does

This skill helps AI coding agents create, edit, debug, review, and harden GitHub Actions workflows and custom actions.

It covers the day-to-day pieces of GitHub Actions work, including:

- workflow syntax
- triggers and filters
- jobs and matrices
- permissions and secrets
- reusable workflows
- custom actions
- deployment environments
- security hardening

## When To Use It

Use this skill when you are:

- working in `.github/workflows/`
- editing `action.yml` or `action.yaml`
- debugging failed GitHub Actions runs
- designing or reviewing CI/CD automation for a GitHub repository

## How It Helps

The skill gives the agent a practical workflow plus a deep syntax reference so it can move from quick fixes to more advanced workflow design without guessing.

## Files

- `SKILL.md` - the agent-facing instructions and workflow
- `references/syntax-reference.md` - deeper GitHub Actions syntax and context details
